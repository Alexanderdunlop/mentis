---
title: Overview
description: Introduction to the MentionInput component and its main features.
---

The `MentionInput` component provides an accessible, customizable input field with support for mentioning users or entities using the `@` symbol. It is designed for chat, comment, and editor interfaces where users need to reference others quickly and intuitively.

**Main Features:**

- Type `@` to trigger a mention modal with selectable options
- Keyboard and mouse navigation
- Customizable options and callbacks
- Accessible with ARIA roles and keyboard support
- Easy integration and styling
