---
title: Installation
description: You can install mentis via npm, yarn, pnpm, or bun.
---

Install the `mentis` package with your package manager of choice.

```shell
npm install mentis
```

```shell
yarn add mentis
```

```shell
pnpm add mentis
```

```shell
bun add mentis
```

[npmjs.com](https://www.npmjs.com/package/mentis)
